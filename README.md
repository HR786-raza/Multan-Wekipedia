## 🏙️ About the Project

**Multan Wikipedia** is a mini informative web project designed to replicate the look and feel of a Wikipedia-style page, highlighting the rich history, landmarks, culture, and importance of Multan in Pakistan's heritage.

It includes multiple sections such as historic places, famous personalities, climate, festivals, and modern development — all presented with a clean, readable layout.

---

## ✨ Features

- 🏰 Detailed content about Multan’s history and culture
- 📷 Images of landmarks and famous sites
- 🧭 Navigation menu for quick section access
- 🌐 Fully responsive design for desktop and mobile
- 📖 Wikipedia-style layout using clean HTML & CSS

---
